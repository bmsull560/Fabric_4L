# Value Fabric Agent Skills Overview

Complete taxonomy of agent skills for the Value Fabric system.

## Tier 1: Knowledge Navigation
- **/graph_traverse** - Navigate Knowledge Graph relationships
- **/semantic_search** - Find entities by meaning
- **/resolve_value_tree** - Build complete Value Trees
- **/find_path** - Find paths between entities

## Tier 2: Reasoning & Inference
- **/multi_hop_reason** - Answer complex multi-hop questions
- **/analyze_gaps** - Identify missing capabilities
- **/detect_conflicts** - Find contradictory information

## Tier 3: Calculation & Quantification
- **/evaluate_formula** - Execute mathematical formulas
- **/sensitivity_analysis** - Run Monte Carlo simulations
- **/aggregate_metrics** - Roll up metrics

## Tier 4: Content Generation
- **/generate_business_case** - Create business case documents
- **/write_executive_summary** - Distill into executive summary
- **/build_narrative** - Create compelling stories

## Tier 5: Research & Enrichment
- **/research_web** - Search web for context
- **/analyze_competitor** - Compare against competitors
- **/enrich_industry_context** - Add industry benchmarks

## Tier 6: Audit & Provenance
- **/trace_provenance** - Show output lineage
- **/audit_confidence** - Identify weak links
- **/verify_sources** - Check source health

## Tier 7: Collaboration & Workflow
- **/escalate_to_human** - Request human review
- **/ask_clarification** - Ask user for missing info
- **/plan_workflow** - Break into sub-tasks

## Tier 8: Meta-Cognitive
- **/self_correct** - Detect and fix errors
- **/quantify_uncertainty** - Express confidence levels
- **/identify_knowledge_gaps** - Recognize missing info

## Specialized Agents

- **@documentation_agent** - Technical documentation and maintenance
- **@business_analyst_agent** - ROI analysis and business case generation
- **@knowledge_navigator_agent** - Knowledge graph exploration and reasoning

## Common Workflows

**Whitespace Analysis:**
`research_web → semantic_search → graph_traverse → analyze_gaps → generate_business_case`

**ROI Calculator:**
`resolve_value_tree → evaluate_formula → sensitivity_analysis → quantify_uncertainty`

**Business Case Generator:**
`multi_hop_reason → enrich_industry_context → build_narrative → write_executive_summary`
