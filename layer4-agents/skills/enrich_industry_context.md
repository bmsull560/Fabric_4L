# Industry Context Enricher Workflow

Use this workflow to add industry-specific context to your analysis.

## Parameters

- `industry`: String (e.g., 'Manufacturing', 'Healthcare') (required)
- `context_type`: BENCHMARKS | REGULATIONS | TRENDS | PAIN_POINTS

## Steps

1. Identify industry segment
2. Retrieve relevant benchmarks
3. Research industry trends
4. Identify regulations if requested
5. Map common pain points

## Output

Industry context data:
- Relevant benchmarks and metrics
- Industry trends summary
- Regulatory requirements (if requested)
- Common pain points for the industry
- Source citations for all data

## Example Use

"What are typical cost reduction benchmarks in manufacturing?"
