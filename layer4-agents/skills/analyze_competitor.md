# Competitor Analyzer Workflow

Use this workflow to compare competitor capabilities against your Value Tree.

## Parameters

- `competitor_name`: String (required)
- `comparison_dimensions`: List (CAPABILITIES, PRICING, POSITIONING, CUSTOMERS)

## Steps

1. Research competitor information
2. Map competitor capabilities to value tree
3. Compare across specified dimensions
4. Identify gaps and overlaps
5. Highlight whitespace opportunities

## Output

Competitive analysis report:
- Competitive positioning matrix
- Capability-to-value mapping comparison
- Gap and overlap analysis
- Whitespace opportunities identified
- Strategic recommendations

## Output

Competitive positioning matrix with whitespace opportunities
