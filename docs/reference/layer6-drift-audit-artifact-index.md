# Layer 6 drift-audit artifact index

## Purpose

Provide a single reviewer-friendly index for Layer 6 wrapper/canonical drift-audit artifacts.

## Canonical artifact location

- `docs/contracts/layer6-audit-artifacts/`

## Start here

- `docs/contracts/layer6-audit-artifacts/README.md`
- `docs/contracts/layer6-audit-artifacts/active-milestones.md`

## Required files for each active milestone

For every milestone declared in `active-milestones.md`, include:

- `docs/contracts/layer6-audit-artifacts/milestones/<milestone-id>/report.md`
- `docs/contracts/layer6-audit-artifacts/milestones/<milestone-id>/checklist.md`

## Required report structure

Milestone reports must follow `docs/contracts/layer6-audit-artifacts/TEMPLATE.md`, including:

- Endpoint inventory
- Frontend usage mapping
- Gateway exposure matrix
- Risk classification
- Owner sign-off block
