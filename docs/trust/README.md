# Trust Center Documentation

This directory contains policy and process artifacts that support Value Fabric's security,
privacy, and vendor governance posture.

## Documents

- `vendor-risk-policy.md` — Internal vendor intake, due diligence, and ongoing monitoring policy.
- `subprocessor-register-template.md` — Template register for tracking subprocessors and transfer details.
- `customer-data-commitments.md` — Customer data ownership, portability, and deletion commitments.
- `coordinated-vulnerability-disclosure.md` — Coordinated vulnerability disclosure (CVD) workflow,
  including scope, SLAs, and safe harbor.
- `internal-risk-register.md` — Internal risk register structure, review cadence, and control ownership.
- `security-trust-summary.md` — Customer-facing trust summary mapped to internal evidence sources.
