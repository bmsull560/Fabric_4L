# Layer 6 drift-audit artifacts

This directory is the canonical location for Layer 6 wrapper/canonical drift-audit evidence.

## Active milestones

Active milestones are declared in `active-milestones.md`.

For each active milestone ID, reviewers and CI expect both of the following files:

- `milestones/<milestone-id>/report.md`
- `milestones/<milestone-id>/checklist.md`

## Template

Use `TEMPLATE.md` as the required section scaffold for every new milestone report.
