# Frontend Development

## Package manager

This frontend uses **pnpm** as the canonical package manager.

- Use `pnpm-lock.yaml` as the single lockfile.
- Do not use `npm install` or commit `package-lock.json`.

## Quick start

```bash
cd frontend
pnpm install
pnpm dev
```

## Common commands

```bash
pnpm check
pnpm test
pnpm build
pnpm exec playwright test
```
