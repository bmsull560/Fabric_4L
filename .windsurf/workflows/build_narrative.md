# Narrative Builder Workflow

Use this workflow to create compelling stories connecting capabilities to business outcomes.

## Parameters

- `starting_point`: Capability or challenge description (string, required)
- `audience`: CFO | CIO | VP_Operations | Procurement (default: CFO)
- `story_arc`: PROBLEM_SOLUTION | TRANSFORMATION | COMPETITIVE_ADVANTAGE (default: PROBLEM_SOLUTION)

## Steps

1. Identify the starting capability or challenge
2. Resolve the value tree
3. Tailor messaging to the target audience
4. Structure the narrative arc
5. Add hooks and evidence
6. Include call-to-action

## Output

Narrative document:
- Structured story following specified arc
- Tailored messaging for target audience
- Key hooks and evidence points
- Supporting data and citations
- Call-to-action recommendations

## Example Use

"Create a story for a CFO about reducing operational costs"
