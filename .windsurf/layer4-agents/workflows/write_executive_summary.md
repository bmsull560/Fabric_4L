# Executive Summary Writer Workflow

Use this workflow to distill complex analysis into a concise executive summary.

## Parameters

- `source_analysis`: Reference to previous agent output (string, required)
- `focus`: ROI | STRATEGIC_FIT | RISK | IMPLEMENTATION (default: ROI)
- `max_length`: Word count (default: 300)

## Steps

1. Retrieve source analysis
2. Extract key findings based on focus
3. Structure executive-friendly format
4. Include bullet points and key numbers
5. Stay within length constraints

## Output

Concise summary with bullet points and key numbers
